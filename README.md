# Custom Gesture Drone

## Goal
Build a custom gesture-controlled micro drone using ESP32-C3.

## Components
- ESP32-C3
- MPU6050
- 720 coreless motors
- DRV8833 motor drivers
- 55mm propellers
- 800mAh LiPo battery

## Progress Log

### Day 1
Researched about the drone basics and learnt the required parts for a mini gesture based drone and ordered them.

### Day 2
Learned ESP8266 basics and created a Wi-Fi control page and hosted a web server locally.
