# ESP32-C3-Based-Gesture-Controlled-Drone

Building a custom ESP32-C3 gesture-controlled micro drone from scratch.

## Custom Gesture Drone

### Goal

Build a custom gesture-controlled micro drone using ESP32-C3.

## Components

- ESP32-C3
- MPU6050
- 720 coreless motors
- DRV8833 motor drivers
- 55mm propellers
- 800mAh LiPo battery
- TP4056 charging module
- JST connectors
